Significance of Landslides Susceptibility Map

Landslides are one of the most widespread and hazardous geohazards in the world. Therefore, predicting landslide susceptibility is an important means of preventing potential natural disasters. Landslide susceptibility mapping based on the environmental factors of the study area not only includes past landslide occurrences as predictive indicators but also has spatial characteristics, allowing the prediction of the hazard levels of landslides in specific locations.

Our research is mainly divided into two parts. First, we conduct landslide susceptibility mapping for the study area. Based on this, combined with the population data of the study area, we perform exposure assessment, categorizing the potential impact of landslides according to population characteristics.

By combining the results with population data, we can closely link the predicted results with the population distribution in the study area, giving the geographically-derived predictions certain social attributes. This integration plays a positive role in policy-making for natural disaster prevention.

Tools and Services

QGIS

A Free and Open Source Geographic Information System. It supports viewing, editing, printing, and analysis of geospatial data in a range of data formats. QGIS has a plugin infrastructure, and provide a lot of data processing tool like GRASS, QuickMapServices.

dzetsaka

Plugin for semi-automatic classification with Gaussian Mixture Model, Random Forest, SVM and K-Nearest Neighbors classifiers. In our project we used Random Forest to do the classifcation.
For more information: https://github.com/nkarasiak/dzetsaka/

OpenLayers

OpenLayers is a JavaScript library for displaying map data in web browsers as slippy maps. It provides an API for building rich web-based geographic applications.It is open-source, provided under the 2-clause BSD License.

GeoServer

GeoServer is an open-source server written in Java that allows users to share, process and edit geospatial data. Designed for interoperability, it publishes data from any major spatial data source using open standards.

FileZilla

FileZilla is a free and open-source, cross-platform FTP application, consisting of FileZilla Client and FileZilla Server. Clients are available for Windows, Linux, and macOS. We use it to publish our WebGIS source to the GeoServer from Politecnico di Milano

GitHub

With GitHub pages, GitHub allows users to host a webpage from users' repository. And we used this service to host this website.
